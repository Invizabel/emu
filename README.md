A script that allows you to download web based emulators and host them locally.

* SWOTGB (Gameboy)
* Javatari (Atari 2600)
* Retroarch (General)
* Web–Øcade (General)
* PlainChess (Chess)
* Classic Minecraft (Minecraft)

To self host: git clone https://github.com/Invizabel/emu && cd emu && python3 setup.py
